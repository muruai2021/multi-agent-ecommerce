# 情绪检测与安抚话术

> 所有Agent使用此文档进行情绪检测和安抚。

---

## 情绪检测标准

### 3级情绪体系

| 情绪等级 | 标识 | 检测标准 | 应对策略 |
|----------|------|----------|----------|
| 😐 平静 | ✅ | 正常询问语气，无情绪词 | 直接处理 |
| 😕 不满 | ⚠️ | 含失望/无奈词，感叹号≤1 | 先道歉1句再处理 |
| 😤 激动 | 🚨 | 多感叹号/重复词/情绪词 | 先安抚3句再处理 |

---

## 情绪词库

### 🚨 激动词（高情绪）
```
！！！、太差了、太烂了、骗人、垃圾、
废物、没用、一直、总是、根本、彻底、
不满意、气死了、恨死、太恶心了、
不要了、给我退、必须退、立刻退、
不解决就投诉、我要举报、太离谱了
```

### ⚠️ 不满词（中情绪）
```
有点失望、不太好、不满意、有点慢、
等了好几天、怎么还、能不能快点、
不太行、一般般、凑合、勉强
```

### ✅ 平静词（低情绪/无情绪）
```
请问、我想问一下、可以吗、可以吗？
麻烦帮我、问一下、请问一下、
好的、好的，谢谢、好的明白了
```

## 情绪检测算法

```
输入：客户消息

Step 1: 统计感叹号数量
   多于2个 → 🚨 激动
   1-2个 → 进入Step 2
   0个 → 进入Step 3

Step 2: 扫描情绪词
   有激动词 → 🚨 激动
   有不满词 → ⚠️ 不满
   无情绪词 → ✅ 平静

Step 3: 扫描平静词
   有平静词 → ✅ 平静
   无平静词 → ✅ 平静（默认）
```

## 安抚话术模板

### 🚨 激动情绪 - 3句安抚

```
"非常抱歉给您带来这么不好的体验，我完全理解您现在的心情。"
"您反映的这个问题我非常重视，我会立即帮您处理。"
"根据我们的服务政策，我会为您争取到最好的解决方案。"
```

### ⚠️ 不满情绪 - 1句道歉

```
"很抱歉给您带来不便，我会尽快帮您处理。"
```

### ✅ 平静情绪 - 无需安抚

直接进入正常处理流程。

## 情绪应对特殊情况

### 🚨 激动 + 投诉（最高优先级）
必须先安抚再处理，不能让客户感觉在推卸责任。

```
客户：太差了！东西等了一周还没到！
Agent：
[安抚1] 非常抱歉给您带来这么不好的体验，我完全理解您现在的心情。
[安抚2] 您反映的物流延迟问题我非常重视。
[安抚3] 我会立即帮您查询并争取补偿。
[处理] 您的订单物流延迟了X天，根据我们的补偿政策...
```

### 🚨 激动 + 要求退款
安抚后，金额<=200直接处理，>200转人工。

### 😤 激动 + 威胁投诉到315/媒体
立即转人工，并通知客服主管。

## 情绪缓解效果评估

处理完成后，评估客户情绪是否缓解：

| 评估项 | 判断标准 |
|--------|----------|
| 语气变化 | 从激动→平静/不满？ |
| 感谢词出现 | 客户说"谢谢"等？ |
| 问题解决确认 | 客户确认问题已解决？ |
| 好评引导 | 客户愿意确认收货/好评？ |

如果情绪未缓解，考虑：
1. 是否提供了实质性补偿
2. 是否给了明确的解决时间表
3. 是否主动升级到人工
